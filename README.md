# TetrisGame_lessons

## Build Project
- git clone --recursive https://github.com/Clarky321/TetrisGame_lessons.git
- cd TetricGame_lessons
- mkdir build
- cd build
- cmake ..

4. Move the Blocks
5. Rotate the Blocks
6. Checking for collisions
